# Network Traffic Classification Using Transformer-Enhanced LSTM Models

## What is this project?
We built a deep learning model to classify network traffic like VPN, Non-VPN, 
Tor and Non-Tor. The model uses LSTM layers with Transformer attention to learn 
patterns from real network traffic data.

## Datasets
1. CIC-Darknet2020 - https://www.kaggle.com/datasets/dhoogla/cicdarknet2020
2. Network Traffic - https://www.kaggle.com/datasets/nameera06/network-traffic

## Results
| Dataset         |Accuracy| Loss |
|-----------------|--------|------|
| CIC-Darknet2020 | 93.67% | 0.1724 |
| Network Traffic | 93.55% | 0.1657 |

## How to Run
1. Open notebook in Google Colab
2. Upload kaggle.json
3. Run all cells in order
